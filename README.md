# Solar Linear Icons for Discourse

A Discourse theme component that replaces the default FontAwesome icons with the beautiful **Solar Linear Icons** set.

## Installation

1. Go to your Discourse Admin panel.
2. Navigate to **Appearance** > **Themes**.
3. Click **Install**.
4. Select **From a git repository**.
5. Paste the URL of this repository: `https://github.com/itsarjunc/discourse-solar-icons`
6. Add the component to your active themes.

## Features

- Replaces core Discourse icons with **Solar Linear** style.
- Consistent and modern aesthetic.
- Fully responsive and supports light/dark modes (uses `currentColor`).
- High-performance unified SVG spritesheet.
